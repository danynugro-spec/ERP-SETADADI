# ERP Penggilingan Padi PRO+ v2.0.0
## CV. SETIA DADI — Single Source of Truth

### Cara Menjalankan
Buka `index.html` di Chrome/Edge — selesai. Offline, tanpa instalasi server.

### Yang Baru di v2.0
- **Master Barang** — satu tabel pusat untuk semua item (gabah, PK, beras, samping)
- **Kategori konsisten** — seluruh modul baca dari `getBarang()` / `getKategoriBarang()`
- **Bug fix mutasi** — Menir/Bekatul = Produk Samping, PK = PK, pembelian beras ≠ Gabah

### Backup & Restore
Menu Administrasi → Backup Enterprise → Download / Restore
